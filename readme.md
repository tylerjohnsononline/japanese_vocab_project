### Why a vocabulary project
I wanted to be able to practice vocabulary words.

This will be a simple program to do vocabulary practice from the command line. 

### how to use
step 1: download this repository

step 2: run the following commands from the cmd(command line: 

```
python -m venv venv_spot
```

step 3: to open the virtual environment, run this command:
```
venv_spot\Scripts\activate.bat
```
step 4: install the requirements
```
pip install -r requirements.txt
```

step 5: run the vocab_test_gui.py file
```
python vocab_test_gui.py
```
step 6: use the window that pops up to test your knowledge of some japanese vocabulary. 

Click new word then show japanese new to see the word in japanese, click show english to show the english word.

To get a new word click new word, then show japanese new, and again show english to reveal the answer.
